# Strategic Consulting Website

A modern, professional website template for a strategic consulting firm.

## Features

✨ **Modern Design**
- Responsive layout optimized for all devices
- Smooth animations and transitions
- Professional gradient backgrounds
- Glassmorphism effects

🎯 **Key Sections**
- Hero section with rotating text
- Services with interactive tabs
- About section with statistics
- Team member showcase
- Client testimonials
- Contact form

⚡ **Interactive Elements**
- Mobile-responsive hamburger menu
- Smooth scroll navigation
- Animated counters
- Tab-based service showcase
- Interactive testimonials

📱 **Mobile Friendly**
- Fully responsive design
- Touch-friendly navigation
- Optimized for mobile devices

## Files

- `index.html` - Main HTML structure
- `tooplate-strategic-style.css` - Complete styling
- `tooplate-strategic-scripts.js` - Interactive functionality

## Setup

1. Clone the repository
2. Open `index.html` in your web browser
3. Replace images in the `images/` folder with your own
4. Customize text content for your company

## Customization

### Change Company Name
Find and replace "StrategicPro" with your company name

### Update Services
Edit the service tabs and details in the Services section

### Add Team Members
Add new team member cards in the Team section

### Modify Contact Information
Update the contact form section with your details
